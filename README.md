\# 基于心理量表分数的焦虑分类 Demo



背景

本项目模拟了 200 名被试的两个心理量表分数（SAS 焦虑自评量表、BAI 贝克焦虑量表），根据总分 > 100 划分为高焦虑组（1）和低焦虑组（0）。旨在演示如何用机器学习方法（KNN 和逻辑回归）对焦虑状态进行分类。



方法

\- 数据生成：正态分布随机数，范围 20\~80。

\- 特征：SAS 和 BAI 分数

\- 标签：0 = 低焦虑，1 = 高焦虑（阈值 100）

\- 模型：

&#x20; - K 近邻（KNN，k=5）

&#x20; - 逻辑回归（Logistic Regression）

&#x20;评估：准确率、分类报告、混淆矩阵



&#x20;结果

\- KNN 准确率：约 0.90

\- 逻辑回归准确率：约 0.88

\- 两个模型均能有效区分高/低焦虑组，KNN 略优（但实际效果取决于数据分布）。
